Dall'elenco seguente è possibile scaricare le presentazioni che abbiamo mostrato alla classe durante le esposizioni delle ricerche.

Le presentazioni sono in formato Microsoft PowerPoint 2003.

  • Windows 98SE, ME, 2000, XP e successivi: per poterle visualizzare è necessario aver installato "Microsoft PowerPoint 2003" o successive versioni. Se "Microsoft PowerPoint 2003" non è installato è possibile scaricare gratuitamente il "Visualizzatore di PowerPoint 2003" cliccando qui.
  • Linux: è sufficiente aver installato il programma gratuito "OpenOffice.org" per visualizzare le presentazioni create con PowerPoint. Se "OpenOffice.org" non è installato, scaricarlo dai repository della propria distribuzione o scaricarlo dal sito ufficiale. Oppure, se sul proprio sistema è installato il programma "Crossover", è possibile installare il "Visualizzatore di PowerPoint 2003" scaricabile da questa pagina anche su Linux.
  • Macintosh: è sufficiente aver installato il programma gratuito "OpenOffice.org" per visualizzare le presentazioni create con PowerPoint. Se "OpenOffice.org" non è installato, scaricarlo da questa pagina. Oppure, se non è possibile installare OpenOffice, è possibile scaricare una vecchia versione del "Visualizzatore di PowerPoint 98" da questa pagina, tuttavia utilizzando questo programma obsoleto le presentazioni non funzioneranno correttamente.
